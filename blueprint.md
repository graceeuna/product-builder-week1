
# Lotto Number Generator

## Overview

A simple, visually appealing web application that generates and displays 6 unique random lottery numbers between 1 and 45.

## Design and Features

*   **Modern Aesthetics:** Clean, centered layout with a modern font and a simple color scheme.
*   **Interactive Elements:** A "Generate" button that, when clicked, produces the lottery numbers.
*   **Number Display:** The generated numbers are displayed in styled circles.
*   **Responsive:** The layout is responsive and works well on different screen sizes.
